App Description

An Android flash card app built with Kotlin. The app displays a question, and when the user taps it, the answer is revealed. Tapping again hides the answer and shows the question again â€” simulating a flash card flip.


Required User Stories

 1. User sees a question displayed on the screen.

 2. User can tap the question to reveal the answer.

 3. User can tap the answer to hide it and show the question again.
